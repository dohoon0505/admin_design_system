02

Sidebar Navigation 사이드바 내비게이션

1차 내비게이션. 섹션으로 그룹핑, 현재 위치를 accent-subtle 배경 + accent 텍스트로 표시.

변형: flat · grouped · nested · collapsed

03

Topbar 상단 바

콘텍스트·전역 액션의 자리. 좌측 제목/브레드크럼, 우측 검색·알림·테마·프로필. 우측 액션은 4개 이하.

홈/회원
04

Breadcrumb 브레드크럼

현재 페이지의 계층 위치. 마지막 항목은 링크가 아닌 텍스트(aria-current).

홈/ 회원/ 상세
05

Page Header 페이지 헤더

콘텐츠 최상단의 제목 블록. 제목 + 설명 + 우측 액션(2차 → 1차). 1차 액션은 우측 끝 1개.

회원 관리

가입 회원을 조회하고 상태를 변경합니다.

06

Tabs 탭

같은 맥락의 뷰 전환. 밑줄형 + 카운트 배지. 클릭해 활성 탭을 바꿔보세요.

변형: underline · segmented · with-count
07

Pagination 페이지네이션

범위 요약 + 페이지 번호 + 이전/다음. 현재 페이지는 accent-subtle, 숫자는 tabular-nums.

1–20 / 128
08

Command Palette 커맨드 팔레트

⌘/Ctrl+K 로 어디서든 열어 명령·페이지를 검색해 즉시 실행. 키보드만으로 완결.

변형: commands · navigation · search · with-recent · grouped
전역 단축키 ⌘K / Ctrl+K 도 동작합니다.

09–20 폼 · 입력

09

Button 버튼

액션의 기본 단위. 위계(1차/2차/ghost/danger) + 크기. 한 화면의 1차 버튼은 하나.

상태: rest · hover · active · focus · disabled · loading 크기: sm · md · lg

한 줄 텍스트 입력. 라벨은 항상 표시(placeholder 대체 금지). 에러는 색 + 문구 + aria-invalid.

Radio 라디오

상호배타적 단일 선택. 2~5개 옵션을 모두 보여줄 때(그 이상은 셀렉트).

17

Switch 스위치 · 토글

즉시 반영되는 on/off. 저장 없이 바로 적용되는 이진 상태에 사용. Space로 토글.

18

File Upload 파일 업로드

19

Form Field 폼 필드 래퍼

42
